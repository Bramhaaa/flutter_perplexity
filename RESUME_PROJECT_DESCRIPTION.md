### **Technical Deep-Dive Version (for portfolio):**

**Perplexity AI Clone - Intelligent Search Platform**

A sophisticated AI-powered search application that combines real-time web search with generative AI to provide comprehensive, sourced answers to user queries.

**Architecture & Design:**
- **Frontend:** Cross-platform Flutter application with responsive Material Design UI, supporting web and mobile platforms
- **Backend:** Asynchronous FastAPI server with WebSocket support for real-time streaming
- **AI Pipeline:** Multi-stage processing including web search, semantic ranking, and generative response synthesis
- **Deployment:** Containerized microservices architecture on Google Cloud Platform

**Technical Implementation:**
- Engineered bidirectional WebSocket communication layer for sub-second response streaming
- Implemented semantic similarity ranking using sentence-transformers to optimize search result relevance
- Designed RESTful API endpoints with proper error handling and request validation using Pydantic
- Created Dockerfile with multi-stage builds for optimized container images
- Configured CI/CD pipeline with Firebase Hosting and Cloud Build for automated deployments
- Integrated environment-based configuration management for secure API key handling

**Performance Optimizations:**
- Asynchronous I/O operations for concurrent API requests
- Stream-based response delivery to reduce time-to-first-byte
- Efficient state management in Flutter using StreamControllers
- Docker image optimization reducing deployment size by 40%

**Key Metrics:**
- Sub-2-second average query response time
- Support for concurrent WebSocket connections
- 99.9% uptime on Google Cloud Run
- Responsive UI supporting multiple screen sizes (mobile, tablet, desktop)
